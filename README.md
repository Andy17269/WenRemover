# WenRemover for macOS

English | [简体中文](README.zh-CN.md)

A SwiftUI-based macOS gadget for batch removal of picture EXIF information.

## Function

- Drag the picture to the application window;

- Select the output directory and rules;

- One-click batch removal of EXIF;

## Run

1. Download the latest version on the Releases page;

2. Install and open the app (you may need to trust this app manually in "Settings");

3. Enjoy!

## Instructions for use

- Drag the picture to the window

- Click "Select Output Directory"

- Click "Remove EXIF"

The output file will be saved to the specified directory according to your custom suffix (customizable, the default is `_clean`).
