# WenRemover for macOS

English | [简体中文](README.zh-CN.md)

A SwiftUI macOS utility for batch-removing EXIF metadata from images.

## Features
- Drag and drop images into the app window
- Choose an output folder
- One-click batch removal of EXIF

## Run
1. Download the latest build from the Releases page
2. Install and launch the app
3. Enjoy!

## Usage
- Drag images into the window
- Click “Choose Output Folder”
- Click “Remove EXIF”

Output files are saved to the selected folder with your custom suffix (default `_clean`).
